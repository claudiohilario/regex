## Metacaracteres 
### Representantes
Metacaractere | Nome | Função
--------------|------|----------
`.`           |Ponto | Representa um caractere qualquer
`[...]`       |Lista | Representa uma lisca com os caracteres permitidos
`[^...]`      |Lista negada| Representa uma lisca com os caracteres não permitidos
